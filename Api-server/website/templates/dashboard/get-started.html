1. Unzip to install
Trên Linux hoặc Mac OS X, bạn có thể giải nén Tee Note từ một thiết bị đầu cuối với lệnh sau. Trên Windows, chỉ cần nhấp đúp vào tee_note.zip để giải nén nó.
$ unzip /path/to/tee_note.zip
2. Upload for Google Chrome
Tệp đuôi .crx tương thích với mọi thiết bị có cài đặt chrome. Bạn chỉ cần mở chế độ nhà phát triển trong phần cài tiện ích rồi kéo thả thư mục đuôi .crx lên trang tiện ích
$ ./tee_note authtoken 21nF6wYCxtmXmNO6kp4agIvM0O4_56h5j5jRTtq6hqPgLH3hd
3. Connect your messenger
Sau khi cài đặt thành công, truy cập vào messenger.com để sử dụng. Hoặc nhấp vào tiện ích phía bên trên thanh tabbar của chrome để mở messenger dưới dạng cửa sổ application desktop
$ ./tee_note connect messenger
4. Fire it up
Đọc tài liệu của tôi về cách sử dụng tin nhắn ghi chú. Hãy thử nó bằng cách chạy nó từ dòng lệnh:
$ ./tee_note help
Để bắt đầu chuyển tiếp đường hầm HTTP tới cổng cục bộ 80 của bạn, hãy chạy bước này tiếp theo:
$ ./tee_note http 80
